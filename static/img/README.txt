Required Images for RecipeHub:

Please add the following images to this directory:

1. hero-image.jpg   - A hero image for the homepage (cooking or food-related)
2. breakfast.jpg    - Image representing breakfast recipes
3. lunch.jpg        - Image representing lunch recipes
4. dinner.jpg       - Image representing dinner recipes
5. dessert.jpg      - Image representing dessert recipes

You can obtain free food images from:
- Unsplash: https://unsplash.com/s/photos/food
- Pexels: https://www.pexels.com/search/food/
- Pixabay: https://pixabay.com/images/search/food/

Recommended image sizes:
- hero-image.jpg: 1200 x 600 pixels
- Category images: 600 x 400 pixels

These images are referenced in the templates and are needed for the application to display correctly.
